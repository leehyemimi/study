속성선택자

속성 선택자(Attribute Selector)는 특정 속성(attribute)을 갖고 있거나 특정 속성이 특정 값 등을 갖고 있는 요소(element)를 선택합니다.

[attributename]

attributename 속성을 가진 요소를 선택합니다. 

h5[class] : class 속성을 가진 h5 요소를 선택합니다.
