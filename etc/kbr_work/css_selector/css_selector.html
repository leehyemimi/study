css 선택자

1. [class^="text_com_"] 'text_com' 로 정확하게 시작하는 요소 선택

  • [ text_com_yellow ]
  • [ text_com_yellow ]
  • [ text_com_yellow ]
  • [ text_com_yellow text_type1]

2. [class$="_yellow"] '_yellow' 로 정확하게 끝나는 요소 선택

  • [ text_com_yellow ]
  • [ text_com_yellow ]
  • [ text_com_yellow ]
  • [ text_com_yellow text_type1]
  • [ text_type1 text_com_yellow] - 추가*

2. [class*="com_"] ‘com_’ 포함하는 요소 선택

  • [ com_ ]
  • [ com_test ]
  • [ test_com_ type1 ]

2. [class|="ico"] ‘ico’ 또는 ‘ico-‘ 으로 시작되는 E를 선택

  • [ ico ]
  • [ ico-test ]
  • [ ico_test ]
  • [ test_ico_ type1 ]
