p[class^="abc"]{background:blue;}

abc 이기만 하면 됨 뒤에 어떤 문자가 달라 붙어도 상관X

가상 요소(Pseudo Element)

: 와 :: 차이는?
::는 css3에서 가상 클래스와 가상 요소를 구분하기 위해 도입되었습니다.
:: 콜론이 2개면 css 안먹힘 가상클래스에서 (-> first-child,last-child,nth-of-type)
가상요소: 요소의 특정 부분을 선택하는 것 (::first-letter,::after...)

  • 나 1번 :first-child (blue)
  • 나 2번 :nth-of-type(2) (red)
  • 나 3번 ::first-letter(50px)
  • 나 4번:last-child(green)
  • ::before 사용 (ul li::before) ->(content:"앞")
  • ::after 사용 (ul li::after) ->(content:"뒤")
