og:image 이미지 설정 정리!

페이스북

큰미리보기 사용하려면 최소 600*315

작은미리보기 600*315 이보다 작을 경우 1:1 비율의 작은 이미지로 표시됨
잘리는부분 없이 표시하려면 최대 1.91:1 비율로 작업해야함

* 페이스북 특이사항 : 이미지는 최대 5개 가능 페이스북 공유시 이미지 선택해서 게시할 수 있음 (그 외 sns는 대표이미지 하나만 노출)

* 주의 : 페이스북에서는 200*200보다 작을시에 디버거 오류가남

카카오톡

이미지 비율을 2:1로 작업해야함 비율이 안맞을시 중간을 기준으로 위아래가 잘림

공통사항

고해상도 기기에서 고화질로 표현 하려면 이미지 최소 1200*630(페이스북기준)
페이스북 1.91:1 비율, 카카오톡은 2:1 비율 고려해서 작업하면됨

특이사항(og:image:width,og:image:height 관련)

og:image:width,og:image:height 값은 필수 요소가 아님
구글검색시 결과 : 비동기적으로 다운로드하고 처리할 필요없이, 즉시 렌더링 하도록 크롤러에 이미지 크기를 지정하는 방법

* 본인결론 : 이미지 태그에 width, height 처리와 같다고 보면 됨
