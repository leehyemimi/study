Font Awesome

1. Font Awesome 이란?

웹 글꼴(Web Font)의 일종 장점: 벡터 그래픽 방식으로 확대해도 깨지지않음 자바스크립트가 사용되지 않기때문에 호환성 문제가 적음 상업적 용도로도 무료로 사용할수있음 크기조절/색상/배경색 바꾸기가 자유로움 단점: ie7 지원안됨[지원하려면 아래링크 삽입(3.2.1버전) 단, 4.3.0버전은 지원하지않음] <!--[if IE 7]>
  <link rel="stylesheet" href="path/to/font-awesome/css/font-awesome-ie7.min.css">
<![endif]-->

2. 적용방법

ㄱ. 웹폰트 사용하는 방법으로 호출
ㄴ. 적용하려는 아이콘을 확인한후 클래스명이나 유니코드값을 확인 - Font Awesome Icons
ㄷ. 아이콘을 태그에 직접 적용하려는 경우 클래스명을 이용하여 아이콘 적용
ㄹ. css를 이용하여 문서에 아이콘을 삽입하려는 경우 :before, :after를 이용하여 content: 값에 유니코드값을 지정

CSS3

최근 이미지를 쓰지 않고 웹폰트를 이용해 아이콘을 표현하는 사이트가 많아졌습니다.
하지만, 몇 개의 아이콘을 쓰기 위해서 많은 아이콘이 들어있는 폰트파일을 불러오는 것도 좋다고만은 할 수 없겠지요.
이미지로 만들지도, 웹폰트를 전부 불러오지 않아도 아이콘을 CSS로 표현할 수 있다면 좋지 않을까요?
